Apple charger utility 
======================
USB charging control utility used to enable/disable charging of an Apple device connected to USB port.

Supported devices:
------------------
* iPad
* iPad2
* iPad3
* iPod Touch 2G
* iPod Touch 3G
* iPod Touch 4G
* iPhone 3GS
* iPhone 4
* iPhone 4S

Requirements:
-------------
Linux, GCC, Make, libusb-1.0
